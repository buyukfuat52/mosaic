# Reproducible Examples

Generate a deterministic synthetic dataset:

```bash
yolo-mosaic synthetic --output-dir examples/synthetic_dataset --num-images 30 --seed 42
```

Create two 2x2 mosaics:

```bash
yolo-mosaic generate \
  --images-dir examples/synthetic_dataset/images \
  --labels-dir examples/synthetic_dataset/labels \
  --output-images-dir examples/outputs/images \
  --output-labels-dir examples/outputs/labels \
  --grid 2 \
  --count 2 \
  --seed 42
```

Render bounding-box previews:

```bash
yolo-mosaic visualize \
  --images-dir examples/outputs/images \
  --labels-dir examples/outputs/labels \
  --output-dir examples/outputs/visualized
```

After running those commands, the expected real output locations are:

- `examples/synthetic_dataset/images/`
- `examples/synthetic_dataset/labels/`
- `examples/synthetic_dataset/classes.txt`
- `examples/synthetic_dataset/summary.json`
- `examples/outputs/images/mosaic_0000.jpg`
- `examples/outputs/labels/mosaic_0000.txt`
- `examples/outputs/visualized/mosaic_0000.jpg`

The files are generated by the implementation and can be regenerated with the same seed.
